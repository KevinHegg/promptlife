Prompt Life v0.26.3

Glossary Dojo Shared Shell Report

Glossary Dojo now uses the shared Play Challenge Foundation shell, progress, feedback, completion, and action components while preserving its 12-question glossary learning loop.

1. Summary

Dojo keeps its quiz engine, glossary content, close distractors, repeat rounds, review rounds, and existing progress store. The UI now sits on the shared Play shell and writes attempt/completion summaries through the shared Play progress key.

  • Wrapped start and question states with PlayChallengeShell and PlayChallengeHeader.
  • Added shared PlayStatusPill and PlayProgressRail for round status and question progress.
  • Replaced answer feedback with PlayFeedbackPanel.
  • Replaced summary lead with PlayCompletionPanel.
  • Updated visible feedback leads to Good distinction. and This choice reveals a common mix-up.
  • Updated summary actions to Try another round and Review missed terms.
  • Bumped the app version to v0.26.3.

2. Recommendations For The Next Pass

  • Move Context Stack onto the same shared shell and completion-panel pattern.
  • Add a compact debug-only Play progress viewer after the storage shape stabilizes.
  • Build Probability Picker before replacing the Attention Match bridge.

3. Challenges And Risks

  • The Dojo engine still owns detailed feedback text; the UI strips older lead phrases and adds the new calm lead.
  • Dojo mastery remains in promptlife.glossaryDojo.v1; shared Play progress records attempt/completion summary only.
  • The existing Vite large-chunk warning remains.
