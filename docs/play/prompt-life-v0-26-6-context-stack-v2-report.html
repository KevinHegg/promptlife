Prompt Life v0.26.6

Context Stack v2 Report

Context Stack is now a three-round shared Play challenge about the context window: visible now is not the same as remembered forever.

typecheck passed build passed pages build passed answer audit passed 320px QA passed 390px QA passed

What Changed

Shared Play Shell

Context Stack now uses the shared Play challenge shell, header, status pill, progress rail, board, choice buttons, feedback panel, action row, scroll hint, token chips, and completion panel.

Three Rounds

  1. Basic context window: old cards fall out when the window fills.
  2. Important detail falls out: keep the goal, constraint, and evidence visible.
  3. Retrieved evidence enters context: retrieval helps only after it is placed in context.

Progress Tracking

Attempts, round progress, completion, review suggestions, best progress, outcomes, and misconception tags now write through promptlife.playChallenges.v1.

Copy Cleanup

Learner-facing Play and Badge copy now says practice challenges, practice set, and coming soon instead of foundation/slate wording.

Learning Design

Core idea: visible now is not the same as remembered forever. The model can use what is inside the current context window; older material can fall out; retrieved evidence helps only if it enters context.
