Prompt Life v0.22

Glossary Dojo V2 Question Design + Results Repair

Date: 2026-06-07

This report wraps the implementation summary, verification results, and representative screenshots for the Glossary Dojo V2 pass.

What Changed

  • Term-to-definition questions now ask What does [TERM] mean? and show definitions only.
  • Definition-to-term questions now show the definition once and use term-only choices.
  • Closest/confusable prompts use plain helper copy: Choose the closest related term.
  • Distractors use learning-path proximity before related-term and global fallbacks.
  • Stored rounds now support exact repeat rounds and missed-question review rounds.
  • The results screen now emphasizes calm round reflection instead of score-like mastery language.
