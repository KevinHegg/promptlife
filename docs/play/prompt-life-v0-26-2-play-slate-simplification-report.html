Prompt Life v0.26.2

Play Slate Simplification Report

A small pass that uses the new Play Challenge Foundation to keep the visible Play page focused on five final or likely-final learning challenges.

typecheck passed build passed build:pages passed audit:answers passed 320px QA 390px QA

1. Summary

The visible Play page now presents exactly five final or likely-final cards: Glossary Dojo, Context Stack, Probability Picker, Prompt Run, and Attention Match. Token Pipeline Relay and How AI Learns remain out of the normal slate.

  • Confirmed the Play page is driven by the shared challenge registry.
  • Simplified each visible card purpose line.
  • Labeled Prompt Run as the capstone challenge and changed its action to Start capstone.
  • Kept Probability Picker disabled as Foundation ready / Coming soon.
  • Kept Attention Match visible while safely reusing the current Attention Weave route/component.
  • Preserved retired compatibility handling for Token Pipeline Relay and How AI Learns.
  • Bumped the app version to v0.26.2.

2. Recommendations For The Next Pass

  • Build Probability Picker as a full decoding challenge for logits, softmax, sampling, and probability-is-not-truth.
  • Replace the Attention Match bridge with a purpose-built token dependency matching activity.
  • Move Context Stack onto the shared Play challenge shell for stronger visual consistency.
  • Add a debug-only local progress viewer after the final Play storage shape settles.

3. Challenges And Risks

  • The pass intentionally preserves old routes and progress bridges instead of deleting legacy Play code.
  • Probability Picker is not playable yet; it is a disabled final-slate placeholder.
  • Attention Match still uses the old Attention Weave interaction underneath.
  • The existing Vite large-chunk warning remains.
