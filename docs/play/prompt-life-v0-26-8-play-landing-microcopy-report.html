This pass removes cryptic practice-move chips from the top Play panel and puts clear practice moves directly on each challenge card.

What Changed

Top Panel

Removed unlabeled name, place, choose, connect, and trace chips. Added: Each challenge practices one model-literacy move.

Practice Moves

Each Play challenge card now includes a clear Practice move: line, such as Name concepts or Choose likely next tokens.

Stats

Active Glossary Dojo progress now reads Current: N correct and Last played: Today. Started but unscored challenges read Current: Started.

Heading Area

Removed the decorative Practice set pill. The heading is simply Play challenges with the helper line Suggested order follows the Journey.

Practice Move Mapping

  • Glossary Dojo: Name concepts.
  • Attention Match: Connect token clues.
  • Probability Picker: Choose likely next tokens.
  • Context Stack: Place context cards.
  • Prompt Run: Trace the loop.
