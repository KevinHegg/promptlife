# Prompt Life App Blueprint

## Audience

Smart high-schoolers, college freshmen, faculty, higher-ed IT leaders, and PhDs outside computer science.

## Through-line

A prompt has a day:

1. **Before Morning** — AI history, training, fine-tuning.
2. **Morning Commute** — text becomes tokens, token IDs, embeddings, vectors, tensors.
3. **Workday** — transformer layers update hidden states using attention and MLPs.
4. **Decision Room** — final hidden state lights up the vocabulary cloud; logits become probabilities; sampling chooses one token.
5. **After Hours** — generation repeats, context windows matter, side tours explain diffusion, multimodal AI, risk literacy, and metaphor limits.

## Teaching pattern

Every screen should answer three questions:

- What is this thing?
- How does it relate to the previous thing?
- What metaphor makes it memorable without lying?

## Mini-games

### Context Stack

Teaches context window, truncation, instruction ordering, recency.

### Attention Weave

Teaches attention as weighted relevance, not consciousness.

### Token Pipeline Relay

Teaches sequence, state, determinism, and small changes compounding.

## Badge

“Prompt Life: Model Literate” means the learner can explain what an LLM is, what it is not, and where real risks live.
