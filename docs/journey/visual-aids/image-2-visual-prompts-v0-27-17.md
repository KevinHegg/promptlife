# Prompt Life Image 2 Visual Prompt Sheet v0.27.17

This sheet is for future Image 2 asset generation only. Do not generate or add PNG assets in this pass. Keep all instructional text, labels, captions, and callouts in HTML so the app remains accessible and editable.

## Production Rules

- Use generated PNGs only for atmospheric, metaphorical, social, or ethical concepts.
- Do not use generated PNGs for exact model mechanisms, token positions, arrows, matrices, axes, context-window boundaries, or numbered step diagrams.
- Generated images must be textless: no readable words, letters, numbers, pseudo-text, UI labels, charts, or document text.
- Use Prompt Life ZenTron Origami style: folded paper, calm academic tone, deep navy/cyan/violet palette, subtle green or amber accents only when meaningful.
- Keep the image mobile-safe at 320px and 390px. Important forms should remain readable when cropped into a wide visual-aid frame.
