# Prompt Life Visual Aid Strategy v0.28.5

This strategy resets the Journey visual-aid plan without redesigning the visuals in this pass. It chooses one recommended path for each of the 39 Journey learning cards.

## Category Rules

- Image 2 Concept Card: polished static conceptual visuals; short embedded text only if human-verified; HTML callouts below.
- Coded Mechanism Mini-Diagram: exact token, context, probability, arrow, or mechanism relationships controlled in HTML/SVG/CSS.
- HTML Comparison Board: 2-4 HTML cards or columns for boundary distinctions; short headings; no SVG text.
- No Visual Needed / Text Callout Only: no visual unless later testing proves one is needed; use text callouts and glossary support.

- Image 2 Concept Card: 12

- HTML Comparison Board: 8
