What Is an LLM?

Stage 1. Before Morning · 2 pilot questions

Learning-objective review

Primary: Reason that an LLM generates response tokens by using learned weights and current context, without implying awareness or perfect search.

Mechanism/boundary: Inference uses fixed learned weights and temporary context to score, sample, append, and repeat next tokens.

Related glossary: llm, prompt, response, token, weight, inference

Nearby distractors: training, retrieval, memory, context window, probability
