## Policy

Generated images should be textless conceptual assets only. They can provide atmosphere, object metaphors, and focal art, but they must not carry instructional text.

All teaching labels, diagrams, arrows, numbered seals, glossary terms, and checkpoint copy should remain HTML/SVG controlled by the app.

## Shared Style

ZenTron Origami:

- Anime clarity.
- Tron glow.
- Japanese paper-cut layering.
- Calming zen garden.
- Rice-paper surfaces.
- Deep indigo ink.
- Cyan/teal/violet/amber computational glow.
- One focal object per image.
- Generous quiet space for app overlays.
- No text.

Forbidden elements:

- No words.
- No letters.
- No UI text.
- No tiny labels.
- No distorted signage.
- No corporate stock-photo look.
- No dystopian skulls.
- No robot overlords.
- No aggressive surveillance wall of screens.
- No dense poster composition.

## Asset Spec: perfect-storm

- ID: `perfect-storm`
- Lesson: The Perfect Storm
- Visual purpose: Show that LLMs arrived through convergence, not one magical breakthrough.
- Prompt for image generation: `Textless ZenTron Origami scene: four luminous rivers of paper and light flow into a calm glowing origami transformer shape at center; rivers suggest data, compute, algorithms, and human labor without labels; rice-paper landscape, faint zen garden contours, deep indigo night glow, cyan and amber edges, anime clarity, Japanese paper-cut layering, serene futuristic learning garden, no text, no letters, no signage.`
- Required style: Calm paper-cut landscape, one central origami transformer object, four distinct unlabeled streams, subtle neon edges.
- Forbidden elements: No storm destruction, no lightning apocalypse, no labels, no corporate data-center stock photo, no robot faces.
- Transparent or simple background preference: Simple mist/rice-paper background; optional transparent-ish edges if export supports it.
- Codex overlay plan: HTML/SVG numbered paper seals and labels: Data, Compute, Algorithms, Human labor; optional small side chip for Incentives.
- Alt text: Four streams of data, compute, algorithms, and human labor flow toward a glowing origami transformer, showing that LLMs emerged from convergence.
- Callouts:
  1. Human-created data.
  2. Compute and storage.
  3. Deep-learning algorithms.
  4. Human labor and incentives.

## Implementation Notes

- Generate images only after the curriculum copy is approved.
- Keep generated images in `public/assets/promptlife/illustrations/` only after explicit asset implementation approval.
- Use app-side alt text, HTML callouts, and SVG overlays.
- Keep diagrams accessible without relying on image color alone.
- If an image generation pass produces text-like marks, reject the asset or crop/remove the marks before use.
- Prefer SVG/HTML diagrams when precision matters; use generated images only for atmosphere or conceptual props.
