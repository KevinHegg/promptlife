Prompt Life v0.6.0 - June 2, 2026

Content Repair and Mobile Shell Report

v0.6 turns Prompt Life into a clearer mobile-first learning path: each lesson now teaches the same way, the app shell scrolls more reliably on phones, the bottom nav has a safe-area-aware frosted gradient, and the review/PDF routes are ready for content QA.

What Is Done

  • Normalized the Journey to the requested 25 lessons, including Prompt vs Response, Vectors, and How AI Learns.
  • Added Brain Bridge to every lesson so brain metaphors include explicit limits.
  • Added reusable visual aids and review routes: /review/lesson-cards and /review/visual-aids.
  • Moved reusable exercises out of Journey lessons so Journey teaches and Play practices.
  • Repaired mobile scrolling, safe-area padding, bottom nav gradient, and route/lesson scroll reset behavior.
  • Added PWA metadata and a lesson-card PDF exporter.

How To Run Locally

npm install

npm run dev

Open http://localhost:5173/.

Verification

npm run typecheck: passed.

npm run build: passed.

npm run export:lesson-pdf: passed.

Responsive smoke check: no horizontal overflow at 320, 390, or 430px; desktop review route renders 25 lesson cards.

Next Three Improvements

  • Add an axe accessibility pass and strengthen the glossary drawer focus trap.
  • Add a small service worker only after the v0.6 content shell is stable enough to benefit from offline caching.
