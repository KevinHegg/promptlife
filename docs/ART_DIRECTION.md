# Prompt Life Art Direction

Use art as visual metaphor, not as a container for dense text.

## Mobile-first rules

- One screen = one idea.
- Text belongs in HTML.
- Images orient the learner emotionally and spatially.
- Avoid slide-deck poster density.
- Use lightweight icons and scene art.

## Core metaphors

- Conveyor belt = tokens.
- Spotlight / arcs = attention.
- Gears / forge = MLP.
- Funnel = softmax.
- Stacked cards = layers and context window.
- Pulse node = seed/run.
- Feature cloud / vocabulary cloud = distributed learned relationships.
- Badge = confidence and literacy.

## Palette

Primary navy, teal, mint, purple, amber, off-white. High contrast required for text.

## Asset folders

- `public/assets/promptlife/brand` — logo and badge.
- `public/assets/promptlife/icons` — small icons.
- `public/assets/promptlife/illustrations` — mobile lesson and game art.
- `design/references` — reference only, not app pages.
