# New Dawn Tiny Interaction Plan v0.25.2

Date: 2026-06-07

## Benefits Worth Taking Seriously

- Interaction type: `benefit-tier-sort`
- Misconception targeted: AI benefits require hype, or benefits make costs irrelevant.
- Expected action: Cycle each claim into a confidence tier.
- Correct feedback: "Insight strengthened. Benefits can be real and bounded when humans review evidence and context."
- Wrong feedback: The learner is told a few cards are still in the wrong group and can keep cycling.
- Status: Implemented now.

## Human-Centered AI

- Interaction type: `human-centered-scenario`
- Misconception targeted: Fluent AI output can replace accountable human decision-making.
- Expected action: Choose the deployment where an AI summary is treated as a draft and a human reviews before action.
- Correct feedback: "Insight strengthened. Human-centered AI keeps responsibility with people and institutions."
- Wrong feedback: "Not quite. Human-centered AI keeps limits, review, and accountability visible."
- Status: Implemented now.

## Better AI Is a Choice

- Interaction type: `better-ai-levers`
- Misconception targeted: AI harms are destiny, or bigger systems are automatically better.
- Expected action: Select levers such as approved-document RAG, sensitive-data protection, source links and review, task-fit model choice, and error monitoring.
- Correct feedback: "Insight strengthened. Better AI is shaped by task-fit design, evidence, privacy, review, monitoring, and governance."
- Wrong feedback: Selected bad levers name the risk, such as uploading private data or removing accountability.
- Status: Implemented now.

## Effective Prompting from Model Literacy

- Interaction type: `prompt-builder`
- Misconception targeted: Prompting is magic wording or permanent training.
- Expected action: Add audience/context, constraints, examples, format, evidence, uncertainty, and review parts to a weak prompt.
- Correct feedback: "Insight strengthened. A good prompt builds useful context for this run; it does not rewrite model weights."
- Wrong feedback: The prompt asks the learner to add the missing context parts.
- Status: Implemented now.

## Model Literate Synthesis

- Interaction type: `synthesis-chain`
- Misconception targeted: Mechanics and human responsibility are separate, or fluency proves wisdom.
- Expected action: Tap the full model story in order from prompt context to human accountability.
- Correct feedback: "Insight strengthened. Model literacy means understanding both the machine chain and the human consequences."
- Wrong feedback: The learner is told which step belongs next and can reset the chain.
- Status: Implemented now.
